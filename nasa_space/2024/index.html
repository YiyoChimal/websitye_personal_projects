NASA Space Apps Challenge 2024 – Earth Observation Data for Agriculture

Objective

Design and develop an Internet of Things (IoT) system that complements satellite-based Earth observation data with real-time ground-level measurements to improve agricultural monitoring and decision-making.

Technologies Used

• KiCad – Electronic schematic capture and PCB design.

• C – Embedded firmware development.

• ATmega328P – Sensor node microcontroller.

• ATmega324P – Gateway controller.

• Microchip Studio – Embedded software development.

• LoRa – Long-range wireless communication.

• Ethernet – Gateway-to-server communication.

My Contributions

• Designed the custom PCBs for both the sensing node and gateway using KiCad.

• Programmed the ATmega328P and ATmega324P microcontrollers in Embedded C using Microchip Studio.

• Implemented the LoRa communication protocol for reliable long-range wireless data transmission.

• Developed the Ethernet communication interface for transmitting sensor data to a central server.

• Integrated the sensing, communication, and embedded control subsystems into a complete IoT platform.

Technical Highlights

• Distributed IoT sensing architecture.

• Custom-designed PCBs for sensor and gateway devices.

• ATmega328P and ATmega324P embedded controllers.

• LoRa-based long-range wireless communication.

• Ethernet connectivity for remote server communication.

• Real-time acquisition of soil moisture, soil pH, temperature, humidity, and atmospheric pressure.

• Modular system designed for precision agriculture applications.

Results

Successfully developed an IoT-based monitoring system capable of measuring soil moisture, soil pH, temperature, humidity, and atmospheric pressure, while reliably transmitting the collected data to a central server using LoRa and Ethernet communication. The project received an Honorable Mention at the regional NASA Space Apps Challenge 2024, demonstrating the effectiveness of integrating embedded systems, wireless communication, and environmental sensing for precision agriculture.
